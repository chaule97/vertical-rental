Rental Sign Integration (OCA)

This module integrates OCA Rental with Sign (sign_oca) to manage
electronic signatures for rental delivery and return workflows.

Features

-  Delivery staff must sign the delivery order before it can be
   validated.

-  Customers sign the delivery document after receiving the rented
   items.

-  Delivery staff must sign the receipt document when accepting returned
   items.

Configuration
=============

Select PDF Reports for Signature

Go to: Settings → General Settings → Rental Sign

Configure:

-  Delivery Report for Signature PDF report used when delivering rented
   items (Outgoing Picking).

-  Return Report for Signature PDF report used when receiving returned
   items (Incoming Picking).

These reports will be rendered and sent to the Sign module for
signature.

Usage
=====

Go to: Sales → Configuration → Rental Signature Template Items

Create one or more template items to define where each signer will sign
on the PDF.

You should typically create:

-  1 template item for Employee signature on Delivery Order

-  1 template item for Customer signature on Delivery Order

-  1 template item for Employee signature on Receipt Document
